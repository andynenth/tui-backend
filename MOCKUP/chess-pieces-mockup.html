Comparison Summary

Chinese Characters

✅ Pros:

  • Authentic traditional xiangqi appearance
  • Culturally accurate and historically correct
  • Familiar to xiangqi players worldwide

❌ Cons:

  • Requires Chinese font loading (SimSun/宋体)
  • Font rendering inconsistencies across platforms
  • Potential scaling issues at small sizes

Production SVG Assets

✅ Pros:

  • Perfect scaling at all sizes (34px-68px+)
  • No font dependencies or loading delays
  • Consistent rendering across all platforms
  • Maintains authentic Chinese character appearance
  • Better accessibility (screen reader support)
  • Smaller file sizes than font loading
  • Organized asset structure for easy import
  • Named to match backend constants

❌ Cons:

  • Requires SVG support (universal in modern browsers)
  • Slightly more complex implementation than fonts
